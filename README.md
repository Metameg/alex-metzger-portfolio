# Alex Metzger — Portfolio v4

IDE-shell aesthetic (VS Code file explorer sidebar, tab bar, status bar)
with professionally styled website content inside the panels.

## Setup

### Add your photo
Drop `profile.jpg` in this folder (portrait 3:4 ratio, 500×650px+).
Without it, an "AM" monogram placeholder appears automatically.

### Add your resume
Drop `resume.pdf` in this folder, or update the href links to a hosted URL.

### Update contact info
In `index.html`, replace:
- `alex.metzger@example.com` → your email
- `linkedin.com/in/alexmetzger` → your LinkedIn
- `github.com/alexmetzger` → your GitHub
- Project `href="#"` → real project/GitHub links

## Navigation
- Click tabs in the title bar or files in the sidebar
- Press **1 / 2 / 3 / 4** to switch panels
- **Cmd/Ctrl+P** → Projects panel
- **Cmd/Ctrl+K** → Contact panel

## Deploy (free)
- GitHub Pages: push repo, enable Pages in Settings
- Netlify: drag-and-drop this folder at netlify.com
- Vercel: connect GitHub repo at vercel.com
